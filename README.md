# VoteIQ - India Election Assistant 🇮🇳

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

> An AI-powered interactive education tool for Indian voters built with Gemini AI.

**[Live Demo](https://voteiq-india.web.app)**

## 🌟 Features
- **Interactive 7-step Election Timeline**: Understand the electoral process step-by-step.
- **AI Chatbot**: Get your election questions answered instantly by an AI assistant.
- **Voter Eligibility Checker**: Quickly check if you are eligible to vote.
- **Election Knowledge Quiz**: Test your knowledge about Indian elections.
- **Hindi + English Support**: Accessible to a wider audience with bilingual support.

## 🛠 Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- **Google Gemini 2.5 Flash API** (Core AI Engine)
- Firebase Hosting
- Google Analytics

## 🚀 How to Run Locally
No build steps or dependencies required!
1. Clone this repository or download the files.
2. Simply open `index.html` in any modern web browser.
